# Public Service Calendar

A calendar of pay days and holidays for federal employees.

```
https://outlook.live.com/owa/calendar/00000000-0000-0000-0000-000000000000/cc5763bf-9777-48f9-9ab7-edb20b932231/cid-F168D51DED2D70DD/calendar.ics
```


## Add to Outlook (Desktop)

Open your Outlook calendar, and on the Home tab, select `Add Calendar > From Internet`, and paste the above URL.

## Add to Outlook (Web)

Follow [Microsoft's Instructions](https://support.microsoft.com/en-us/office/import-or-subscribe-to-a-calendar-in-outlook-com-or-outlook-on-the-web-cff1429c-5af6-41ec-a5b4-74f2c278e98c) using the above URL to add the calendar.
